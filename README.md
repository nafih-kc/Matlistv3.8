# Matlistv3.8

🚀 MatList Pro
MatList Pro is a smart materials and grocery list app designed for Indian tradesmen (electricians, plumbers, carpenters) and everyday households.
It’s fast, simple, and built for real-world usage — no unnecessary typing, no clutter.
✨ Features
🛒 Smart Grocery Lists
Predefined items for quick selection
Supports Manglish-friendly usage
🔌 Materials List for Workers
Electricians, plumbers, carpenters
Saves time on repetitive list creation
⚡ Super Fast UI
No lag, no heavy frameworks
Runs smoothly even on low-end devices
📱 Install as App (PWA)
Add to home screen via browser
Works like a real app (<1MB size)
📄 PDF Export
Download lists as PDF
Compatible with Chrome, Samsung Browser, WebView
🐔🐟 Meat & Fish Category
Extended grocery support for Indian users
🎯 Why MatList Pro?
Typing full lists in Notes apps is slow and annoying.
MatList Pro solves this by:
Giving ready-made items
Reducing typing effort
Making list creation almost instant
🛠️ Tech Stack
HTML
CSS
JavaScript (Vanilla)
👉 Built as a single-file app for simplicity and portability.
📦 Installation
Option 1: Use Online
Open the app in your browser.
Option 2: Add to Home Screen (Recommended)
Open in Chrome
Tap 3-dot menu
Select "Add to Home screen"
Done — it works like an app
